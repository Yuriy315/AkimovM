## Описание проекта

Это простое GUI-приложение на Python, которое помогает бороться с прокрастинацией и разнообразить рутину. Оно случайным образом выбирает для вас задачу из предопределённых категорий (Учёба, Спорт, Работа) или из вашего собственного списка.

Приложение обладает следующими возможностями:
*   **Генерация случайной задачи:** Получите порцию мотивации одним кликом.
*   **История задач:** Все сгенерированные задачи сохраняются и отображаются в виде списка.
*   **Фильтрация:** Просматривайте историю только по интересующей вас категории.
*   **Добавление собственных задач:** Расширяйте базу задач под свои нужды.
*   **Сохранение состояния:** История и ваши задачи сохраняются в `JSON` и загружаются при следующем запуске.
*   **Валидация ввода:** Защищает от добавления пустых или дублирующихся задач.

## Технологии

*   **Язык:** Python 3.13
*   **GUI:** `tkinter` (стандартная библиотека)
*   **Случайные числа:** `random`
*   **Работа с данными:** `json`
*   **Контроль версий:** `Git`

## Как запустить

1.  Убедитесь, что у вас установлен Python 3.9 или новее.
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/[ваш_логин]/random-task-generator.git
    cd random-task-generator
    ```
3.  Запустите приложение:
    ```bash
    python main.py
    ```
4.  (Опционально) Создайте виртуальное окружение для изоляции зависимостей.

## Примеры использования (тесты)

1.  **Тест генерации:** Выберите категорию «Спорт» и нажмите «Сгенерировать задачу». В поле отобразится случайное задание (например, «Сделать зарядку»).
2.  **Тест истории:** Сгенерируйте 3-4 задачи. Они должны появиться в списке истории ниже.
3.  **Тест фильтрации:** Выберите фильтр «Учёба» в списке истории. В списке должны остаться только задачи, связанные с учёбой.
4.  **Тест добавления задачи:**
    *   Введите «Помыть посуду» и выберите категорию «Работа». Нажмите «Добавить». Задача должна появиться в выпадающем списке категорий.
    *   Попробуйте добавить пустую задачу — должно появиться предупреждение об ошибке.
5.  **Тест сохранения:** Закройте приложение, откройте его снова. История задач должна сохраниться.
